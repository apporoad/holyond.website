## 所谓的圣光状态
圣光是指神圣的时光，指的是思考到一定阶段出现的事情，我们通过不断地思考，可以发现解开一切的钥匙。
但，圣光是很难到达的，这需要个人不断对思考的努力，不仅要思考有意义的事情，还需要让思考可以无限化、不中断。
然后还需要找到美，沉淀自己的价值观，放飞思考本身。
